# LogLite

A simple log server that accept http POST request to save log, and displays logs in web UI.

# Major Dependencies

* Node.js
* Sqlite3
* React
* Material UI
* Express

Please check package.json for other dependencies.

# Installation and Testing

* Install
`npm install`

* Build
`npm run build`

* Start server
`npm run start`

* Test Send Post
`node test_post_log.js`
